

# Portfolio Website for Avanish Kumar Dwivedi

## Overview
A clean, modern, and minimal Product Designer portfolio website that highlights your technical CS background as a unique advantage. The site will be recruiter-friendly, mobile responsive, and structured around detailed case studies.

---

## Pages & Sections

### 1. Home / Hero
- Full-screen hero with your name, role ("Product Designer with a Technical Edge"), and a brief one-liner about combining CS + design
- Two CTA buttons: **View Work** (scrolls/links to case studies) and **Contact Me**
- Subtle animated background or geometric accent elements for visual interest

### 2. About Section
- Split layout: text on one side, a professional photo placeholder on the other
- Short narrative about your CS (AI & ML) background and transition into Product Design
- Four strength cards with icons:
  - User-Centered Thinking
  - Product Problem Solving
  - Developer-Friendly Design
  - Interaction & 3D Experience Design

### 3. Case Studies Section
Three detailed case study cards on the main page, each linking to a full dedicated page:

**Case Study 1: Math4Kids**
- Structured sections: Problem → Research → User Flow → Wireframes → High-Fidelity UI → Key Decisions → Outcome
- Image placeholders for each stage
- Clean step-by-step narrative layout

**Case Study 2: Sticky Notes**
- Structured sections: Problem → Feature Planning → User Journey → Low-Fi to High-Fi → Clean UI → Prototype
- Before/after style comparisons with image placeholders

**Case Study 3: 3D Interactive Website**
- Structured sections: Goal → Tools Used → Interaction Design → Implementation → Impact
- Emphasis on Spline/Dora tooling and engagement metrics

Each case study page will use a consistent template with a progress indicator, large image areas, and clear typography.

### 4. Skills Section
- Grid of skill pills/cards with subtle hover effects
- Categories: User Research, Wireframing, Prototyping, Interaction Design, Design Systems, Responsive Design, Frontend Feasibility

### 5. Tools Section
- Icon-based display of tools: Figma, Webflow, Spline, Dora, Canva
- Clean horizontal layout with tool logos/icons

### 6. Contact / Footer
- Simple contact section with email link, LinkedIn link, and a Resume download button
- Minimalist footer with copyright

## Design Direction
- **Color palette**: Neutral base (white/off-white background, dark text) with one accent color (e.g., a muted blue or indigo) for CTAs and highlights
- **Typography**: Clean sans-serif, large headings, comfortable reading sizes
- **Layout**: Generous whitespace, card-based components, smooth scroll transitions
- **Mobile**: Fully responsive — stacked layouts on mobile, hamburger navigation

## Navigation
- Fixed top navbar with links: Home, About, Work, Skills, Contact
- Mobile hamburger menu
- Smooth scroll for single-page sections, separate routes for case study detail pages

## Technical Notes
- No backend needed — fully static/frontend-only
- Placeholder images throughout (you can replace with real screenshots later)
- All content hardcoded based on your brief
